## 📱 项目简介

KarmaPath 是面向印度市场的寺庙祈福与命运探索 App。

### 核心功能
- 🏛️ **寺庙发现** - 印度教神庙、锡克教谒师所、佛教圣地
- 🙏 **祈福点** - 许愿石、许愿树、供奉处、圣水池
- 🔮 **吠陀占星** - Kundali 星盘、每日运势
- 📖 **神话故事** - Ramayana、Mahabharata
- 💬 **灵验口碑** - 信徒还愿故事

### 技术栈
- Android Only (97% Android 市场份额)
- Unity + C#
- APK < 15MB
