## 📱 项目简介

KarmaPath 是面向印度市场的寺庙祈福与命运探索 App，基于"不认命"产品理念，深度本地化为印度文化。

### 核心功能
- 🏛️ **寺庙发现** - 印度教神庙、锡克教谒师所、佛教圣地
- 🙏 **祈福点** - 许愿石、许愿树、圣水池、神龛、朝圣路线
- 🔮 **吠陀占星** - Kundali 星盘、每日运势、吉日选择
- 📖 **神话故事** - Ramayana、Mahabharata 地方版本
- 💬 **灵验口碑** - 信徒还愿故事分享
- 🗺️ **朝圣路线** - 跨邦朝圣规划与导航
